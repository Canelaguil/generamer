\subsection{Detailed Description}
Simple Post\+Script graphics interface. 

\begin{DoxyVerb}Sets up printing to the given file. If ``fp`` is omitted,
``sys.stdout.buffer`` or ``sys.stdout`` is assumed.
\end{DoxyVerb}
