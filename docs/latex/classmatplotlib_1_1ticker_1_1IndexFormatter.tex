\subsection{Detailed Description}
\begin{DoxyVerb}Format the position x to the nearest i-th label where ``i = int(x + 0.5)``.
Positions where ``i < 0`` or ``i > len(list)`` have no tick labels.

Parameters
----------
labels : list
    List of labels.
\end{DoxyVerb}
