\subsection{Detailed Description}
\begin{DoxyVerb}Like `property`, but also triggers on access via the class, and it is the
*class* that's passed as argument.

Examples
--------
::

    class C:
        @classproperty
        def foo(cls):
            return cls.__name__

    assert C.foo == "C"
\end{DoxyVerb}
