\subsection{Detailed Description}
\begin{DoxyVerb}Context manager for `.ioff`.

The state is changed in ``__init__()`` instead of ``__enter__()``. The
latter is a no-op. This allows using `.ioff` both as a function and
as a context.
\end{DoxyVerb}
