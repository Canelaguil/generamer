Map tag numbers to tag info. 

id\+: (Name, Type, Length, enum\+\_\+values)

The length here differs from the length in the tiff spec. For numbers, the tiff spec is for the number of fields returned. We agree here. For string-\/like types, the tiff spec uses the length of field in bytes.
