\subsection{Detailed Description}
\begin{DoxyVerb}Context manager for `.ion`.

The state is changed in ``__init__()`` instead of ``__enter__()``. The
latter is a no-op. This allows using `.ion` both as a function and
as a context.
\end{DoxyVerb}
